# hetzner-robot

AI-generated OpenAPI definition for the hetzner-robot API: https://robot.hetzner.com/doc/webservice/en.html

To install dependencies:

```bash
bun install
```

To run the generation of hetzner-robot-openapi.json:

```bash
bun run index.ts
```
